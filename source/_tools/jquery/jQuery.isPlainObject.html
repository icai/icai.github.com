返回值:BooleanjQuery.isPlainObject(obj)

概述

测试对象是否是纯粹的对象（通过 "{}" 或者 "new Object" 创建的）。

参数

objObjectV1.4

用于测试是否为纯粹的对象

示例

描述:

测试是否为纯粹的对象

jQuery 代码:
jQuery.isPlainObject({}) // true
jQuery.isPlainObject("test") // false
